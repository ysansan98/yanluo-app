# Windows 热键监听验收清单

## 0. 环境准备

- [ ] 在 Windows 机器执行 `pnpm run keyboard:build:win`
- [ ] 执行 `pnpm run keyboard:prepare`
- [ ] 启动应用 `pnpm dev`
- [ ] 确认已配置快捷键（例如 `Ctrl + Z`）

## 1. 核心触发（P0）

- [ ] 按下并保持快捷键，语音会话开始（`onPress` 路径）
- [ ] 松开快捷键，语音会话结束（`onRelease` 路径）
- [ ] 连续触发 10 次，无崩溃、无明显卡顿

## 2. 修饰键与别名（P0）

- [ ] `LeftCtrl + Z` 可触发
- [ ] `RightCtrl + Z` 可触发
- [ ] `LeftAlt + Space` 可触发
- [ ] `RightAlt + Space` 可触发
- [ ] `LeftShift/RightShift` 组合均可触发
- [ ] `Win` 相关组合按配置行为正确（若配置了 Win/Meta）

## 3. 快捷键热更新（P0）

- [ ] 在设置中把快捷键从 A 改为 B，无需重启立即生效
- [ ] 改为 B 后，A 不再触发
- [ ] 更新快捷键后无“粘键”现象（按键状态正确复位）

## 4. 预过滤行为（P1）

- [ ] 非目标按键频繁输入时，不触发语音流程
- [ ] 目标组合中的任一键事件可正常上送并驱动匹配
- [ ] 切换快捷键后，旧组合的按键事件不再影响触发判断

## 5. 边界场景（P1）

- [ ] 锁屏再解锁后可恢复触发
- [ ] 睡眠唤醒后可恢复触发
- [ ] 应用窗口失焦/切换程序后仍可全局触发
- [ ] 快速连按（抖动输入）不会重复进入异常状态

## 6. 回归检查（P1）

- [ ] `pnpm run lint` 通过
- [ ] `pnpm run typecheck` 通过
- [ ] `pnpm run test:unit` 通过
